# ft_transcendence

Full-stack multiplayer game platform (42 project).

## Structure

- **`apps/`** — Microservices and frontend. Each app has its own README and docs.
- **`docs/`** — Project documentation.
- **`monitoring/`** — Prometheus & Grafana.
- **`logging/`** — Logstash and related config.

For setup, API, and service details, see the README and `docs/` in each app under `apps/`.
